Market Mayhem - Real Phone Version

This version runs from this computer and lets phones join on the same Wi-Fi.

Same Wi-Fi start:

1. Install Node.js LTS from https://nodejs.org/ if it is not already installed.
2. Double-click start-game.bat.
3. Keep the Market Mayhem Server window open while you play.
4. The host screen opens on this computer.
5. On phones, type one of the join links shown on the host screen.
6. Players enter names on their phones.
7. The computer host picks 5, 8, 12, or 15 weeks.
8. The computer host clicks Start Game.

Host screen:
http://127.0.0.1:3000/host.html

Phone screen:
Use the Wi-Fi join link shown on the host screen.

Internet start:

1. Double-click DOUBLE CLICK ME - START INTERNET GAME.bat.
2. If it says cloudflared is missing, let it install with winget or install Cloudflare Tunnel from https://developers.cloudflare.com/tunnel/.
3. Run DOUBLE CLICK ME - START INTERNET GAME.bat again after cloudflared is installed.
4. Keep the internet window open while you play.
5. Send friends the link shown under SEND THIS TO FRIENDS.

The internet link changes each time you start internet mode.
The internet link also gets saved to INTERNET-JOIN-LINK.txt.

End of week:

The host clicks Resolve Week.
The host screen shows public Week Results.
Each phone gets a private Week Report with income, expenses, asset changes, events, status, and Pay Bills / Skip Bills.

Phone layout:

Players use tabs for Market, Cars, Housing, Inventory, and Report.
Owned cars and houses have Manage screens so jobs, tenants, upgrades, repairs, and selling are not all mixed together.

Stop:

Close the Market Mayhem Server window.
